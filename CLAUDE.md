# 每日口說練習框架

## 核心規格

- **時長**：25 分鐘 + 90 秒回饋
- **劑量**：8 句／日
- **配額**：12 個高頻內容詞【HF】＋ 3 個高頻詞組【BUNDLE】

## 標註系統

| 符號         | 意義       | 範例                                     |
| ------------ | ---------- | ---------------------------------------- |
| `/`          | 語塊邊界   | In my VIEW / the main POINT              |
| `大寫`       | 核重音     | but the deTAILS MATter                   |
| `_`          | 連音       | done_it, near_the                        |
| `【HF】`     | 高頻內容詞 | 【HF】REDUCE                             |
| `【BUNDLE】` | 高頻詞組   | 【BUNDLE】One of the main challenges is… |

## 三遍法

1. **慢速清晰**：重音位置正確，語塊邊界不亂
2. **正常語速**：弱讀、連音到位
3. **錄音**：僅允許一次修正，不重來

## 時間結構

| 時段      | 內容                                             | 句數 |
| --------- | ------------------------------------------------ | ---- |
| 0–3 min   | 校準句（每日固定同一句）                         | 1    |
| 3–9 min   | 音段對立（如 /iː/–/ɪ/、/θ/–/ð/）                 | 2    |
| 9–15 min  | 超音段焦點對比（A/B 兩種讀法）                   | 2    |
| 15–20 min | 連續語流＋語篇模板（縮約、連結、邏輯詞）         | 2    |
| 20–25 min | 語篇組織＋結論收束（although/however/therefore） | 2    |
| 最後 90s  | 回饋                                             | —    |

## 回饋三欄（每欄最多 2 點）

| 可理解度阻礙點     | 可修正機制 | 明日最小調整 |
| ------------------ | ---------- | ------------ |
| 哪個音／節奏出問題 | 具體練法   | 明天怎麼改   |

## 進階任務

**韻律任務**：每句只選一個 HF 詞承擔核重音，其餘降為次重音或弱讀

**替換任務**：保留句型骨架，只替換 HF 詞（如 reduce → decrease / cut），維持相同語塊切分與語調

## 高頻詞來源

- **內容詞**：NGSL / Oxford 3000-5000 / BNC-COCA 1k–2k
- **詞組**：常見 lexical bundles（The evidence suggests… / One of the reasons is… / It depends on…）

## 每日範例格式

見 `example-day*.md` 檔案，包含：

- 今日主題
- HF 清單（12 個）
- BUNDLE 清單（3 個）
- 8 句完整標註
- 韻律任務指定
- 替換任務建議
